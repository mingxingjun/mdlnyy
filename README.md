# 🎨 绘梦应答录——玛德蕾娜的颜料宇宙

> *从「应援站」到「绘梦互动宇宙」的华丽蜕变*

这不再是冰冷的资料站，而是玛德蕾娜的 **「数字写生本」** ——她的档案是写生手稿，技能是颜料实验记录，时装是设计稿。粉丝们可以翻阅她的创作思路，下载她的设计素材，还能实时"对话"让她用颜文字和绘画比喻回应你。

## ✨ 项目定位

- 🎨 **玛德蕾娜的数字写生本**：她的档案、技能、时装都以手绘风格呈现

## 🌟 核心功能模块

### 🖼️ 绘梦档案库（角色展示）
- **写生本风格**：模拟她的手绘档案页，技能介绍做成"颜料工具解析卡"
- **颜料泡涌特效**：首页Banner带有分层渐变模拟的动态效果
- **涂鸦便签**：语音台词以散落便签形式展示，hover播放软萌语音

### 💬 应答涂鸦本（AI聊天系统）
- **颜料画布设计**：聊天框模仿画布，消息如颜料滴，回应带颜文字
- **绘师思维AI**：植入"绘师思维"，常用色彩、绘画比喻回应
- **彩蛋触发**：生日、时装相关内容触发隐藏语音彩蛋
- **多平台支持**：OpenAI、Claude等多个AI平台，UI设计为"颜料罐切换器"

### 🎪 创作工坊（社区互动）
- **同人绘稿墙**：粉丝作品以"漂浮颜料泡泡"形式展示
- **AI点评彩蛋**：点击作品可获得她的AI点评回应
- **调色盘素材包**：壁纸、头像设计成手绘设计稿风格

## 🧪 技术亮点：让技术也染上"颜料味"

### 前端设计
- **颜料流体响应**：页面缩放时元素像颜料般自然流动重组
- **颜料物理动效**：按钮点击→颜料溅落涟漪，页面跳转→画卷展开收起
- **创作氛围材质**：画布粗糙肌理背景，颜料管磨砂质感按钮

### AI聊天系统
- **绘师人格训练**：不仅复刻台词，更植入"绘师思维"和色彩比喻
- **涂鸦日记存储**：聊天记录自动生成带日期天气插画的日记
- **视觉反馈系统**：对话时屏幕边缘飘落颜料花瓣，发送按钮为颜料管挤压样式

## 🎨 自定义指南

### 更换角色主题

1. **颜色主题**：修改 `css/style.css` 中的CSS变量
```css
:root {
    --primary-color: #667eea;    /* 主色调 */
    --secondary-color: #764ba2;  /* 辅助色 */
    --accent-color: #f093fb;     /* 强调色 */
}
```

2. **角色信息**：修改各HTML文件中的角色相关内容
   - 角色姓名、出处、生日等基础信息
   - 角色故事时间轴
   - 名台词和名场面
   - 应援口号

3. **图片素材**：
   - 角色立绘：建议尺寸 800x1200px
   - 场景截图：建议尺寸 1920x1080px
   - 头像图片：建议尺寸 200x200px
   - 壁纸素材：提供多种尺寸

### 添加图片素材

由于这是静态页面模板，图片路径已预留，你需要：

1. 创建 `images` 文件夹
2. 添加以下类型的图片：
   - `character/` - 角色相关图片
   - `moments/` - 名场面截图
   - `community/` - 社区作品
   - `wallpapers/` - 壁纸素材
   - `avatars/` - 头像素材

3. 更新HTML文件中的图片路径
